The origin Brackish dataset only contains videos, so we recommend downloading the dataset from Tianchi and using the provided script to extract frames for easier usage.  
Please download Brackish Dataset from this link:https://tianchi.aliyun.com/dataset/93749  
The downloaded source files include the following contents:  
```
├──script  
    ├──frameExtractor.py
    ...
├──dataset  
└──annotations  
```
step 1:Use frameExtractor.py to extract video frames.  
step 2:organize the directory structure as shown below:  
```
├──Brackish  
    ├──train  
    ├──test  
    ├──val  
    ...
```
